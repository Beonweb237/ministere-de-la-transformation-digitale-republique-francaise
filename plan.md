# Plan : Site Web Gouvernemental Ministériel Haut de Gamme

## Objectif
Créer un site web gouvernemental premium, riche en fonctionnalités et en contenu, exploitable par un ministère (services publics, annonces officielles, e-services, portails thématiques).

## Stages

### Stage 1 — Design & Architecture
- Charger le skill `vibecoding-webapp-swarm` pour le workflow de développement web.
- Charger le skill utilisateur `web-template-library` pour les templates gouvernementaux/institutionnels.
- Créer le design PRD (design.md) avec l'identité visuelle gouvernementale (couleurs sobres, doré/bleu marine, tons chauds, élégance institutionnelle).
- Définir l'architecture des pages et des sections.

### Stage 2 — Dévelopment
- Générer les assets (images, icônes) nécessaires via les outils d'image.
- Développer les composants React avec Next.js, Tailwind, shadcn/ui.
- Implémenter les pages : Accueil, Actualités, Services, Ministère, Contact, E-services.
- Intégrer animations, transitions, et effets visuels premium.

### Stage 3 — Build & Deploy
- Compiler et builder le projet.
- Déployer sur l'environnement de production.

## Architecture des Pages
1. **Accueil** — Hero slider, actualités en vedette, services rapides, chiffres clés, discours officiel, agenda
2. **Actualités** — Liste d'actualités avec filtres, articles détaillés
3. **Services** — Catalogues des services publics avec recherche
4. **Le Ministère** — Biographie du ministre, organigramme, missions, textes fondateurs
5. **Contact** — Formulaire de contact, coordonnées, FAQ
6. **E-Services** — Portail des démarches en ligne

## Identité Visuelle
- Palette : Bleu marine profond (#0A2463), Or/ambre (#D4A574), Blanc cassé (#F5F0EB)
- Typographie : Elégante, lisible, institutionnelle
- Style : Épuré, solennel, moderne, avec des touches dorées subtiles
