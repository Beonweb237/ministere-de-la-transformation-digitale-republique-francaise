# Plan V2 : Adaptation Thème Africain + Corrections

## Problèmes identifiés
1. **Chevauchement header/hero** : Le header transparent chevauche le titre hero "Services publics:" (visible sur capture)
2. **Liens de navigation trop simples** : Besoin d'un mega menu riche et structuré
3. **Thème français** : Doit être adapté pour un contexte africain

## Thème Africain
- **Pays** : République Fédérale d'Afrique Centrale (RFAC) — fictif mais crédible
- **Ministère** : Ministère de la Transformation Digitale et de l'Innovation Publique
- **Couleurs** : Bleu institutionnel profond + accents terre cuite/or savane/vert brousse
- **Images** : Architecture gouvernementale africaine, personnages africains, paysages savane/brousse, réunions communautaires
- **Textes** : Adaptés au contexte administratif africain

## Agents Parallèles

### Agent 1 : Navbar + Mega Menu + Layout/Footer (african-navbar)
- Corriger le chevauchement header/hero (z-index, padding)
- Créer un mega menu riche avec 5 colonnes structurées :
  - **Services** : Particuliers, Entreprises, Collectivités, Associations, International
  - **Démarches** : État civil, Identité, Aides sociales, Fiscalité, Création d'entreprise
  - **Actualités** : Communiqués officiels, Événements, Consultations, Revue de presse
  - **Politiques Publiques** : Programmes nationaux, Rapports annuels, Partenariats internationaux
  - **Transparence** : Budget national, Marchés publics, Organigramme, Open Data
- Adapter le thème africain dans Navbar/Footer/Layout (logo, couleurs, textes)
- Adapter le Footer avec coordonnées africaines

### Agent 2 : Home + About (african-pages-1)
- Regénérer les images hero et modules avec thème africain
- Adapter tous les textes (hero, services, chiffres, actualités)
- Adapter About (ministre africain, historique, équipe)

### Agent 3 : Services + Démarches + Actualités + Contact (african-pages-2)
- Adapter Services (catégories africaines)
- Adapter Démarches (procédures administratives africaines)
- Adapter Actualités (actualités africaines)
- Adapter Contact (coordonnées africaines)

### Agent 4 : Politiques + Transparence (african-pages-3)
- Adapter Politiques (programmes nationaux africains)
- Adapter Transparence (budget, contrats, organigramme africain)
